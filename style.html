styleetiketi

Bir belge için stil bilgilerini tanımlar. Style (Stil), dosyası birçok tasarımın içerisinde gördüğümüz içerisinde CSS tasarım parametreleri (kodları) içeren bir dosyadır. Yeni nesil web sitelerinin stil şeması olarak adlandırabileceğimiz bu stil dosyası CSS komutları yardımıyla tüm web sitesini şekillendirmeye yardımcı olur.
